Tclspec

Tclspec is a feature rich Behaviour Driven Development framework for the Tcl programming language. It is heavily inspired by RSpec and other BDD tools.

Synopsis

# spec/lsearch_spec.tcl
describe "lsearch -exact" {
    it "returns -1 if the given element can not be found" {
        expect [lsearch -exact 4 [list 1 2 3]] to equal -1
    }
}

tclspec spec/lsearch_spec.tcl
